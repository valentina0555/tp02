Preguntas

1-¿Que es una URL?

Una URL (Uniform Resource Locator) es una cadena de texto que especifica dónde puede estar una página,texto, video,etc.Es también conocido como enlance.

2-¿Que es una URL absoluta?

Una URL absoluta es una dirección web, incluye el protocolo, dominio, y la ruta del archivo. Esta URL se usa para enlazar a recursos en otros sitios web

3-¿Que es una URL relativa?

Una URL relativa es un tipo de enlace

4-¿Cuál es la diferencia de una URL relativa y una URL absoluta?

La diferencia entre una URL absoluta y relativa, es que, en la relativa,tiene la información necesaria para encontrar un recurso. Y la relativa encuentra información mediante una URL absoluta.

6-¿Que es un directorio?¿Que relación hay con una carpeta?

Un directorio es un contenedor en el que se almacenan una agrupación de archivos informaticos y otros subdirectorios, atendiendo a su contenido. directorio y carpeta es lo mismo.

7-En la estructura de un proyecto web, ¿a qué se llama directorio raíz?¿Con qué símbolo se representa?

El directorio raíz es la carpeta del disco duro que corresponde a este espacio de nombres de dirección URL. Por ejemplo, si se coloca un archivo llamado file. htm en la carpeta raíz de la aplicación web, se obtiene una dirección URL disponible en http://example/file.htm.

El simbolo que representa el directorio raíz es la barra "/"

10-¿Cuál es el propósito de la etiqueta anchor?

El propósito de la etiqueta anchor es crear hipervinculos.Los "hipervinculos" es un elemento de documento HTML que vincula a una parte del documento o a un documento HTML. Las etiquetas que incluyen el atributo "href" son para generar enlaces que te dirigen hacia una pagina.

11-¿Que atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

El atributo que se utiliza en la etiqueta anchor es "href" que contiene la URL.

12-¿Que sucede si el atributo "href" en la etiqueta anchor está vacío?

Si el atributo "href" está vacío, no te lleva a ningún lado, no te aparece nada

13-¿Cuál es el uso del atributo "target" en la etiqueta anchor y cuáles son los posibles valores que puede tener?

El uso de la etiqueta "target" es que los usuarios van a ser redirigidos con el hipervinculo
